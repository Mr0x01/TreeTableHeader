# 树形表格表头生成器
## 说明
- jQuery仅用到了选择器，可以仅用原生JS。
- 数据放入array中，array[0]为第一列、array[1]为第二列，以此类推。
```
var array = [
            ['人类','兽人','暗夜'],
            ['男', '女'],
            ['盗贼', '法师', '猎人', '术士','牧师']
        ];
```
